
Wood & Co. - Static e-commerce demo
----------------------------------
- Static site built with HTML, Bootstrap, and plain JavaScript.
- Currency uses Philippine Peso (₱) as requested.
- Cart supports: Add, Remove, Update quantity, Clear, and persists in localStorage.
- No backend or database — demo only.
How to use:
- Open index.html in a browser (or serve the folder with a static server).
- products.html lists items with 'Add to Cart' buttons.
- cart.html shows cart contents and allows quantity updates and removal.
